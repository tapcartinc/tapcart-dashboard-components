A ThemeProvider component is a wrapper that easily allows you to adjust the styling of the layout of components particularly on the marketing website.
The theme prop takes predesigned options that correlate with Tapcart branding. This is important for efficient development and consistent styling.
The current theme options are base, invert(base), offWhite, neutral, gradient(foregroundColor, backgroundColor). The cool thing about using the ThemeProvider is the styling that is adjusted easily based on the theme that is passed to the ThemeProvider component.


<br />


## ThemeProvider Example:

```jsx
 <ThemeProvider theme={invert(base)}>
     <H1>Header</H1>
     <P1>Header</P1>
     <Button>Button</Button>
 </ThemeProvider>
```
