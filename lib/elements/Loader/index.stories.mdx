## Quick Start
To create an Loader component, import the <pre>`Loader`</pre> component and give it the appropriate props (reference below).
You can adjust the size and colors it takes manually. Occassionally a loader will be used inside of a parent div with a background color. For this, add the prop `parentBgColor` so the `Loader` can
style accordingly. When a loader is shown inside of a `Button` component there are no extra props that need to be added; simply add the correct props for the Button component and use state for the isLoading prop inside of Button.


## Fields Example:

```jsx
<Loader loaderColors={{fg: '', bg: ''}} size={number} />
```
