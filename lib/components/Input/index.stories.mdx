## Quick Start
To create an InputV2 component, import the <pre>`InputV2`</pre> ...


## Fields Example:

```jsx
  <InputV2
    label="Input label"
    onChange={setInput}
    value={state.input || ""}
    placeholder="Enter Input Text"
    name="input"
  />
```
