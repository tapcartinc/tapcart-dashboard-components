The <pre>Fields</pre> component consists of a parent container within a form with a list of <pre>Fields</pre> as children.

<em>Examples of where this is used in the Merchant Dashboard:</em>

* -Settings/AccountSettings
* -Most of Settings...

<br />

## Quick Start
To create an Fields component, import the <pre>`Fields`</pre> component and the <pre>`Field`</pre> component as the parent component, Fields, children.
Fields component must have children but they don't necessarily have to be a <pre>Field</pre> component (reference Submission Steps page on Merchant dashboard).


## Fields Example:

```jsx
  <FieldsV2 sectionHeader="Section Header">
        <FieldV2>
          <FieldHeader stretch={false}>
            <FieldLabel>Label here</FieldLabel>
            <FieldDescription>Description here</FieldDescription>
          </FieldHeader>

          <Toggle
            className="flex-end"
            type="basic"
            id="toggle"
            toggleState={state.toggleState}
            onChange={() =>
              setState((prevState) => ({
                ...prevState,
                toggleState: !state.toggleState,
              }))
            }
          />

          <Input

            value={state.test || ""}
            placeholder="Enter Popup Text"
          />
        </FieldV2>
        <FieldV2>
          <FieldHeader stretch={false}>
            <FieldLabel>Label here</FieldLabel>
            <FieldDescription>Description here</FieldDescription>
          </FieldHeader>

          <SegmentedControl
            type="secondary"
            items={{
              disabled: "disabled",
              optional: "optional",
              required: "mouse",
            }}

            current={state.currentSelection}
          />
        </FieldV2>

        <FieldV2
          {...props}

          label="Display sold out products"

          <FieldHeader stretch={true}>

          </FieldHeader>
          <ImageUploaderV2
            label="Image"

          />

```
