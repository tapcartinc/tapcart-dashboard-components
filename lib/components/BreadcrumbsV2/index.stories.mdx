# BreadcrumbsV2

---
<br />
<Story id="examples-components-breadcrumbsv2--basic" />
<br />

The Breadcrumb component is a parent container that wraps a list of BreadcrumbItem components.
This component and the BreadcrumbItem child can receive it's own designs, classNames, style in the outside application.

<em>Examples of where this is used in the Merchant Dashboard:</em>

* -Integrations details page
* -Submit steps page

<br />

## Quick Start
To use the Breadcrumbs component, use import the `Breadcrumbs` component; this component must include BreadcrumbItem components.
Each BreadcrumbItem handles its own state for whether the current crub is active or not active.

## BreadcrumbsV2 Example:

```jsx
const MyAccordion = () => (
  <BreadcrumbsV2>
    <BreadcrumbItemV2>
      <a href="/#">
        <Icon type="design" iconLeft />
      </a>
    </BreadcrumbItemV2>
    <BreadcrumbItemV2>
      <a href="/#">crumbies</a>
    </BreadcrumbItemV2>
    <BreadcrumbItemV2>
      <a href="/#">crumbies</a>
    </BreadcrumbItemV2>
  </BreadcrumbsV2>
);
```
