## Quick Start
To create an Button component, import the <pre>`Button`</pre> component and give it the appropriate props (reference below).


## Fields Example:

```jsx
<Button
  color={color}
  onClick={changeColor}
  isLoading={false}
  isDisabled={false}
  thin={false}
  plainText={false}
  onChange={() => setToggle(!toggle)}
>
  children
</Button>
```
