# PopoutContainerV2

---
<br />
<Story id="examples-components-popoutcontainerv2--basic" />
<br />

This component is a popout container that is triggered on hover. It is used frequently throughout our dashboard.
Each `PopoutContainerV2` must include a `PopoutItem`. The `PopoutItem` component is what wraps the item that will
control the hover trigger. `PopoutContainerV2` must also include `PopoutOptions`. `PopoutOptions` will include a
list of options, `PopoutOptionV2`, each having their own set of props and click handlers.

<em>Examples of where this is used in the Merchant Dashboard:</em>

* -Design blocks
* -Collections drag and drop parents

<br />

## Quick Start
To use the PopoutContainerV2, import `PopoutContainerV2`, `PopoutItem`, `PopoutOptions`, and `PopoutOptionV2` components.


## PopoutContainerV2 Example:

```jsx
      <PopoutContainerV2 direction="right">
        <PopoutItem>
          <ContainerBox
            customHeight={200}
            customWidth={300}
            customWidthMd={100}
            bg={colorState}
          />
        </PopoutItem>
        <PopoutOptions>
          <PopoutOptionV2
            id={1}
            onClick={() => console.log("edit")}
            icon="edit"
            label="edit"
          ></PopoutOptionV2>
          <PopoutOptionV2
            id={2}
            onClick={() => console.log("edit")}
            icon="edit"
            label="edit"
          ></PopoutOptionV2>
        </PopoutOptions>
      </PopoutContainerV2>
```
