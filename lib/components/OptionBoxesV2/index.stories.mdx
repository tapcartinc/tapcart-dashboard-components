# OptionBoxesV2

---
<br />
<Story id="examples-components-optionboxesv2--basic" />
<br />

The `OptionBoxesV2` component needs children of `OptionBox`. The functionality for allowing multiple option selection vs one selection at a time is to be handled outside of these components but both options are doable.
Reference the storybook file for this component to see an example of setting the `selectionOptions` for an OptionBoxesV2 component (both multi option selection and one option selection).

<em>Examples of where this is used in the Merchant Dashboard:</em>

* -Engage page
* -Menu dropdown somewhere

<br />

## Quick Start
To create a OptionBoxesV2 component, import the `OptionBoxesV2` and `OptionBox` components.


## PopoutContainerV2 Example:

```jsx
    <OptionBoxesV2 selectedOptions={state.selectedOptions}>
      <OptionBox value="product" onClick={clickOption} id={1}>
        <Icon type="product" />
        <Sofia>Product</Sofia>
      </OptionBox>
      <OptionBox value="push" onClick={clickOption} id={2}>
        <Icon type="push" />
        <Sofia>Push</Sofia>
      </OptionBox>
    </OptionBoxesV2>
```
