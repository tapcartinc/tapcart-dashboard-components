## Installing

You can install the Tapcart Dashboard Component Library by adding this in your dependencies

```
"tapcart-dashboard-components": "github:tapcartinc/tapcart-dashboard-components#master"
```

- NPM: `npm install tapcart-dashboard-components -D`
